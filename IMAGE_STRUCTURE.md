- **Style**: Clean, modern, premium
- **Content**: Realistic phone frame with app interface

- **Style**: Natural, authentic, diverse
- **Content**: African/Muslim students in learning environments
- **Mood**: Positive, engaged, focused
- **Diversity**: Various ages, genders (respectfully covered)

- **Style**: Traditional, elegant, respectful
- **Content**: Geometric patterns, calligraphy, architecture
- **Quality**: Vector format preferred for scalability
- **Colors**: Gold, emerald, ivory palette

## ğŸ“ Usage Guidelines

### **Cultural Sensitivity**
- Ensure all Islamic imagery is respectful and appropriate
- Avoid images with people in inappropriate dress
- Use authentic Islamic art and architecture
- Respect Arabic script and Quranic verses

### **Technical Requirements**
- **Format**: JPG for photos, SVG for icons/patterns
- **Resolution**: Minimum 1200px width for main images
- **Optimization**: Compress for web without quality loss
- **Naming**: Use descriptive, consistent file names

### **Brand Consistency**
- Maintain golden hour/warm lighting theme
- Use consistent color palette (emerald, gold, ivory)
- Ensure all images support the luxury, spiritual brand
- Balance modern technology with traditional Islamic elements
